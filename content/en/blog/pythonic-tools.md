## 通过pymusic-dl下载音乐
```zsh
pip install pymusic-dl  # 安装音乐模块
music-dl -k "<音乐名>"   # 下载音乐
```

### 通过you-get下载视频

```zsh
pip3 install you-get
python3 -m pip install --upgrade you-get
you-get "url"  # 下载url内的视频
```
